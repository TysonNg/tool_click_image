# ✅ Kiểm Tra Nhanh: Tại Sao Không Chạy Được?

## 🔧 Các vấn đề phổ biến

### ❌ "Cửa sổ đích không còn hợp lệ"

**Nguyên nhân:**
- [ ] Cửa sổ game đã bị **đóng**
- [ ] Cửa sổ game đã bị **tối thiểu hóa**
- [ ] Bạn **alt-tab** sang ứng dụng khác
- [ ] Bạn **chuyển focus** sang cửa sổ khác

**Cách khắc phục:**
1. ✅ Đảm bảo game window **ĐANG MỞ**
2. ✅ Đảm bảo game window **KHÔNG BỊ TỐI THIỂU HÓA**
3. ✅ Đảm bảo game window **CÓ FOCUS** (là cửa sổ đang hoạt động)
4. ✅ Bấm **"Chọn Cửa Sổ Đích"** lại
5. ✅ **Giữ game window ở phía trước**
6. ✅ Bấm **"Chạy"**

---

### ❌ "Cửa sổ đích bị mất trong quá trình chạy"

**Nguyên nhân:**
- [ ] Bạn **alt-tab** trong khi scenario đang chạy
- [ ] Bạn **tối thiểu hóa** game trong khi scenario đang chạy
- [ ] Bạn **đóng** game trong khi scenario đang chạy

**Cách khắc phục:**
1. ✅ Chờ scenario **hoàn thành hoặc bấm STOP**
2. ✅ Mở game lại (nếu đã đóng)
3. ✅ Bấm **"Chọn Cửa Sổ Đích"** lại
4. ✅ Bấm **"Chạy"** lại
5. ✅ **ĐỢI** cho đến khi scenario hoàn tất - **KHÔNG ALT-TAB**

---

## 🎯 Quy trình chuẩn

```
1. Mở AutoClick
   ↓
2. Bấm "Chọn Cửa Sổ Đích"
   ↓
3. Chọn cửa sổ game (nó sẽ được highlight)
   ↓
4. Chế độ "Có Focus" - Click vào game window
   ↓
5. ✅ Game window là cửa sổ ACTIVE (đang có focus)
   ↓
6. Bấm "Chạy"
   ↓
7. ⏸️ ĐỢI cho đến khi hoàn tất
   ↓
✅ Xong!
```

## 🚨 Lưu ý quan trọng

| ✅ NÊN LÀM | ❌ KHÔNG NÊN LÀM |
|-----------|-----------------|
| Giữ game window mở | Đóng game |
| Giữ game window không tối thiểu hóa | Tối thiểu hóa game |
| Giữ game window ở trạng thái active | Alt-tab sang ứng dụng khác |
| Chọn lại cửa sổ nếu có lỗi | Chạy cùng lúc |
| Chờ scenario hoàn tất | Click đi click lại |

## 💡 Mẹo

**Nếu bạn chỉ muốn MonitorClick (không chạy kịch bản):**
- Bấm "Capture Coordinates" bình thường, không cần giữ game active

**Nếu bạn muốn chạy kịch bản:**
- Hãy chuẩn bị sẵn game window trước
- Không bấm bất kỳ phím/chuột nào trong khi scenario chạy
- Nếu cần dừng → bấm "STOP"
