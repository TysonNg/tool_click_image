Step 1: Install Python packages
  $ pip install pyautogui pywin32

Step 2: Run the application
  $ python autoclick_standalone.py

Step 3: Use it!
  - Enter window title (e.g., "Notepad")
  - Click "Find Window"
  - Click "Capture Relative Position"
  - Move mouse to target and press ENTER
  - Click "Test Click"
  - Done! ✅

  ✅ Capture coordinates RELATIVE to the window (not screen!)
  ✅ Coordinates work even when you move the window
  ✅ Click at precise positions
  ✅ Support pixel and percentage coordinates
  ✅ Professional GUI with real-time feedback
  ✅ Works across multiple monitors

KEY DIFFERENCE FROM OTHER TOOLS:
  Most tools capture SCREEN coordinates:
    - Save position (1920, 1080)
    - Move window
    - Position is now WRONG ❌

  AutoClick Pro uses WINDOW-RELATIVE coordinates:
    - Save position (100, 150) inside window
    - Move window anywhere
    - Position is still CORRECT ✅
