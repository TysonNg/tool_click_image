Retry logic:
1. Force garbage collection: gc.collect()
2. Brief pause: time.sleep(0.1)
3. Try operation: os.rename(old, new)
4. If fails with PermissionError, retry (up to 3x)

Fallback:
1. Copy folder to temp location
2. Delete original folder
3. Rename temp to final name
